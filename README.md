# Orsell's Extended Entities

A Portal 2: Community Edition addon repository for a set of custom-made AngelScript entities that enhance the Hammer mapping experience, as well as providing QoL improvements.

## EVERYTHING HERE IS STILL A WIP

Current Implemented Entities:

* `prop_faithplate`: A faith plate entity that combines the usual faith plate setup with collision and model, but adds some more customization and control. DOES NOT CURRENTLY HAVE ANIMATIONS!
